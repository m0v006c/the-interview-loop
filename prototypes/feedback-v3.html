How an experienced candidate would walk through this
Short version: grid traversal — for every unvisited land cell, BFS/DFS to mark all connected land, then increment. O(m × n).

Code (Python, BFS)

from collections import deque

def numIslands(grid):
    if not grid or not grid[0]: return 0
    rows, cols = len(grid), len(grid[0])
    count = 0
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == '1':
                count += 1
                q = deque([(r, c)])
                grid[r][c] = '0'
                while q:
                    i, j = q.popleft()
                    for di, dj in ((-1,0),(1,0),(0,-1),(0,1)):
                        ni, nj = i+di, j+dj
                        if 0 <= ni < rows and 0 <= nj < cols and grid[ni][nj] == '1':
                            grid[ni][nj] = '0'
                            q.append((ni, nj))
    return count

Complexity

Time: O(m × n) · Space: O(m × n)

Common pitfalls

  • Claiming O(N log N) — no sorting happens.
  • Forgetting to mark visited → infinite loop.
  • DFS recursion on huge grids → stack overflow.
